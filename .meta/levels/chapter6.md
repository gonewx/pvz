# 生存模式 (Survival Mode)


**第五章（屋顶）已经是《植物大战僵尸》PC原版冒险模式的最后一章**。

在您击败关卡5-10的僵尸博士后，冒险模式的主线剧情就已经全部完成了。游戏会播放经典的结尾曲MV《Zombies on Your Lawn》，并给予您银向日葵奖杯。

之后，您可以重复游玩冒险模式，或者进入**迷你游戏 (Mini-Games)、益智模式 (Puzzle) 和 生存模式 (Survival)** 进行挑战。这些模式没有章节划分，而是一系列独立的关卡合集。

因此，这里我将为您详细介绍**生存模式**，因为它在结构上与冒险模式的场景划分最为相似，可以看作是冒险模式的“下一章”或“终极挑战”。

**核心机制**: 玩家需要在同一个场景中，连续抵御一波又一波越来越强大的僵尸进攻。每防守住一面“旗帜”（通常是2波僵尸），游戏会暂停，让玩家有机会调整下一轮要使用的植物卡片。场上已种植的植物会保留。

## **生存模式：前院白天 (Survival: Day)**
*   **场地布局**: 5行草地，白天场景。
*   **难度分级**:
    *   **普通 (Normal)**: **共5面旗帜**。难度会平缓上升，僵尸种类与第一章类似，但数量和密度会增加。
    *   **困难 (Hard)**: **共10面旗帜**。难度极高，后期会出现所有章节的僵尸混合进攻，包括巨人僵尸等。
*   **核心挑战**: 在有限的资源下，构建一个能够应对连续5轮或10轮猛攻的、可持续发展的强大阵型。

## **生存模式：前院黑夜 (Survival: Night)**
*   **场地布局**: 5行草地，夜间场景，有墓碑，无自然阳光。
*   **难度分级**:
    *   **普通 (Normal)**: **共5面旗帜**。
    *   **困难 (Hard)**: **共10面旗帜**。
*   **核心挑战**: 经济压力巨大。需要在黑暗中高效地产出阳光，同时应对墓碑带来的场地限制和后期刷出的僵尸。对阳光菇和向日葵的布局要求非常高。

## **生存模式：后院泳池 (Survival: Pool)**
*   **场地布局**: 6行场地，白天泳池。
*   **难度分级**:
    *   **普通 (Normal)**: **共5面旗帜**。
    *   **困难 (Hard)**: **共10面旗帜**。
*   **核心挑战**: 水陆两栖作战的终极考验。需要同时兼顾6条线路的防守，尤其是水路会出现潜水、海豚、冰车等多种高威胁单位，防线极易被突破。

## **生存模式：后院雾夜 (Survival: Fog)**
*   **场地布局**: 6行场地，雾夜泳池，无自然阳光。
*   **难度分级**:
    *   **普通 (Normal)**: **共5面旗帜**。
    *   **困难 (Hard)**: **共10面旗帜**。
*   **核心挑战**: 结合了黑夜的经济压力、泳池的多线作战和浓雾的视野限制，是游戏中最复杂的场景之一。玩家不仅要管理经济和防线，还要持续用路灯花或三叶草来维持视野，对操作和策略的要求极高。

## **生存模式：屋顶 (Survival: Roof)**
*   **场地布局**: 5行倾斜的屋顶。
*   **难度分级**:
    *   **普通 (Normal)**: **共5面旗帜**。
    *   **困难 (Hard)**: **共10面旗帜**。
*   **核心挑战**: 必须使用投手类植物，且每个格子都需要花盆，使得阵型构建成本高昂且缓慢。后期面对大量的巨人僵尸和投石车僵尸，防线极易被远程摧毁或被近战突破。

## **终极挑战：生存模式：泳池（无尽）(Survival: Pool (Endless))**
*   **场地布局**: 6行场地，白天泳池。
*   **波数/旗帜**: **无限**。
*   **核心挑战**: 这是《植物大战僵尸》的终极考验。僵尸的种类、数量、密度和血量会随着旗帜数的增加而无限提升，后期会出现满屏幕的红眼巨人僵尸（血量为普通巨人的两倍）。这个模式考验的是玩家能否构建出理论上最完美、最高效、能自我修复的“永动机”阵型（如经典的“八炮流”、“冰瓜阵”等），并挑战自己的极限。
